# richyuh

Personal website and blog

## Stack

- [Astro](https://astro.build/) - Static site generator
- JavaScript/HTML/CSS

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

The static site will be generated in the `dist/` directory.

## Preview

```bash
npm run preview
```
